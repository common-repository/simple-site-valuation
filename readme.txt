This plugin will allow you create an badge showing your site's valuation and/or some key seo data ... more here: http://domof.com/p/help.html/

== Description ==

By using this plugin you'll be creating a badge that will display your domain's current estimated value in dollars, Google page rank, and Alexa popularity ... usefull for small seo campaigns.
These key seo values or simply the site valuation will convince your visitors of the work amount you've put onto your domain.
Once created, the image badge (.png) will self update when your seo page report at domof.com will be updated.

== Installation ==

A. Use the Simple Site Valuation badge as a Widget
1. Extract and copy the folder 'simple-site-valuation' to the '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu from WordPress wp-admin
3. Go to Appearance -> Widgets page, drag and drop the widget to one of the sidebars
4. If any, follow the steps shown on the widgets panel, save and enjoy ;-)

B. Use the Simple Site Valuation badge on posts article editor
1. Extract and copy the folder 'simple-site-valuation' to the '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu from WordPress wp-admin
3. When editing an article (post), simply select one text link and press the 'Simple Site Valuation' button to convert one article text link into one seo badge
4. If any, follow the steps shown on the article popup, insert and enjoy ;-)
